The Confluence editor is what you'll use to create and edit Confluence pages, blog posts, and comments. You can enter content as you would in a Word document, apply formatting, and embed other content and files on the page. Confluence supports a number of useful formatting features:

The basics

Text formatting

The basics include text formatting such as bold, italic, underlined, strikethrough, and inline code formatting. Text colour, for example green or red, would be nice.
